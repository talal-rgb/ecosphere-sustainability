For EU Companies

If CSRD applies, you must report under ESRS. Layer GRI for broader impact storytelling if stakeholders demand it. Use ISSB-aligned disclosures for global investors. TCFD is now integrated into ISSB/ESRS.

For US Companies

SEC climate rules require TCFD-aligned disclosures. Use SASB for industry-specific metrics. Consider ISSB for global consistency. GRI is optional but useful for comprehensive reporting.

For Global Companies

Align with ISSB as the global baseline. Add CSRD/ESRS for EU operations. Use GRI for impact materiality where relevant. Map all frameworks to avoid duplicate work — Terrnix can help.

For SMEs

Start with GRI Standards (core option) for simplicity. If CSRD-listed SME, use ESRS LSME standards (simplified). Focus on material topics only. Consider SASB if investor-facing.
